Utility.css
A clean set of tools.

Tiny tools to get you going. I put together a set of interesting utilities that can be used while creating a website. Just with pure CSS, and compatible with most frameworks around. All the tools used have liks to the original project, which in some cases, is way more complete than this. I just used some snippets that may be of interest. No "beautifying", no nothing. Just tools.
(Actually, there is a very small amount of "beautifying" elements, but just for basic presentation of the page. I consider that important, and no page should ever be published without some very basic elements like fonts and such. That's my opinion though...)


Download

Check out the source code on GitHub, or download it here.



Usage

Include utility.min.css in your project using:
<link rel="stylesheet" href="utility.min.css">

Gotchas

  • This stylesheet DOES use some Bootstrap inspired elements. It's not my goal to make a Bootstrap clone, but I must be humble and use what I think it's simple and fantastic about it!
  • It should be fully compatible with Bootstrap too, and any other Framework. Despite that, it's not 100% guaranteed, and if you find any inconsistencies, please open an issue.
  • There are 3 types of responsiveness here: If nothing is said, it's the default one that applies to all screens. But there is also Tablet (t:), for devices between 480 and 750px width, and mobile (m:) for devices with less than 480px. Basically all you have to do is to append a t: and/or m: (without any spaces) with the class you want to assign to those devices to (only on some features, this document will let you know which ones!)

Another Grid

Originally from Andrei Neculaesei's Simple.css. It was a bare-boned (and awesome) responsive CSS Framework, from which I took the responsive grid. Here's how it works:

The basic idea is that you build a grid with up to twelve collumns. To build it you use this format: .w1/3, which means it occupies one third of the parent element's total width. You can use w1/2, w2/4, etc. to make a 50% collumn. Any version of the mathematical combinations will work, as long as it adds up to 1 (1/1, 2/2, 3/3, and so on).

Ok, so let's say you want three rows. The first will have 3 columns, all equal. The second will have 2 columns, with the first having 3/5 of the total width. And the third will be divided in half:

w1/3
w1/3
w1/3
w3/5
(no size inserted)
w1/2
w3/6
  <div class="block">
    <div class="line">
      <div class="sep w1/3">w1/3</div>
      <div class="sep w1/3"> w1/3</div>
      <div class="sep w1/3">w1/3</div>
    </div>
    <div class="line">
      <div class="sep w3/5">w3/5</div>
      <div class="sep ">(no size inserted)</div>
    </div>
    <div class="line">
      <div class="sep w1/2">w1/2</div>
      <div class="sep w3/6">w3/6</div>
    </div>
  </div>

NOTE: On the second line, I didn't need to fill the last collumn, because it will auto-ajust to fit.
On the last line, 1/2 = 3/6 = 0.5. So you can use that combination.

This grid can also be configured to respond differently to different sizes of screens. There's the normal size, from 750px width and up. Tablet size, from 480 to 750px. And Mobile (less than 480px). Tablet and Mobile sizes can be accessed by appending t: or m: to your desired display for those devices. Ex: sep w1/3 t:w1/2 m:1/1. This would produce a collumn that would take one third of the screen on a computer, a half on a table, and the full width on a smartphone.

Video Wrapper

Originally from CSS Tricks. It will allow you to have responsive video, so you won't lose either the ratio of your videos, nor the mobile size of your page when viewed on a small device.

You just need to use the .videoWrapper class, and inside it, and one of the following: <iframe>, <object>, <embed>.

Code Highlighter

Yes, the code highlighter you've been looking at is also part of this Toolkit! Cool, huh?

This is a tweak I made to IAmCreative's Code Highlighter. You can check the original work here. It's made in pure CSS. No JavaScript (so no actual syntax highlight), so it's incredibly light.

Extremely easy to use, just call the <pre> element and assign the .highl class to it. If you want a dark theme, append .dark. If you want a title (the language you are using perhaps?), append data-code="language" too. Easy!

Here are some examples, with the code being the necessary line to call it:

  <pre class="highl" data-code="HTML">
      

  <pre class="highl dark" data-code="HTML">

NOTE: If the background lines are not properly aligned with the code, just edit the line with pre.highl { background-position: 0 -1px; } (change the -1px to another value).



Other Elements:

  • Just some simple text alignment elements for you: .text-left, .text-right, .text-center, .text-justify.
  • Some simple element alignment elements: .left, .right, .center.
  • .disp-block will change the display of an element to block.
  • Use the .block to create a simple container for your content. It works almost like Bootstrap's .container.
  • Use the <code> tag to type code, commands, and that sort of stuff.
  • Use .round make the corners round on elements that don't have it.
  • Use .visible and .invisible to show or hide elements. It also allows you to show and hide elements based on the device used. Ex. t:invisible m:visible will hide the element on a Tablet, but show it on a Computer and Smartphone (you don't need to state that the computer is .visible).
